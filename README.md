# Instachords

Instachords is a simple tool that generates chord progressions to inspire your next musical creation. Whether you're a songwriter facing writer's block or a producer looking for fresh ideas, Instachords provides instant musical inspiration.

## How to Use

1. Visit [Instachords webpage](https://danielmroczek.github.io/instachords)
2. Select your preferred instrument
3. Click "Generate New Progression"
4. Play through the suggested chords
5. Keep generating until you find inspiration!

## Development

### Prerequisites

- Modern web browser
- Basic understanding of HTML, CSS, and JavaScript

### Local Setup

```bash
# Clone the repository
git clone https://github.com/danielmroczek/instachords.git

# Navigate to project directory
cd instachords

# Open in your browser
open docs/index.html
```
